Aplicație pentru gestionarea organizării evenimentelor

# Descriere

Este o aplicație realizată în C# care permite gestionarea și organizarea diferitelor tipuri de evenimente, precum nunți, aniversări, botezuri, petreceri sau conferințe. Aplicația oferă utilizatorilor posibilitatea de a administra informațiile despre evenimente într-un mod simplu și eficient.

# Informații gestionate pentru fiecare eveniment

Pentru fiecare eveniment sunt stocate următoarele date:

-numele evenimentului

-tipul evenimentului (nuntă, aniversare, botez, petrecere, conferință etc.)

-organizatorul evenimentului

-data și ora desfășurării

-locația evenimentului

-numărul estimat de invitați

-bugetul estimat al evenimentului

# Funcționalități principale

Aplicația oferă următoarele operații de bază:

-adăugarea unui eveniment nou

-editarea informațiilor unui eveniment existent

-ștergerea unui eveniment

# Scopul aplicației

Scopul aplicației este de a facilita organizarea și evidența evenimentelor, oferind o modalitate eficientă de gestionare a informațiilor și de prevenire a conflictelor de programare.
